# Alma on Abbott - Landing Page

A responsive landing page for Alma on Abbott luxury living development, built with HTML and Tailwind CSS.

## Setup Instructions

### Build CSS with Tailwind CLI

This project uses Tailwind CLI standalone executable for optimal performance (1.7KB vs 3.5MB CDN).

**Build CSS once:**
```bash
./tailwindcss-windows-x64.exe -i ./src/input.css -o ./dist/output.css --minify
```

**Watch mode (auto-rebuild on changes):**
```bash
./tailwindcss-windows-x64.exe -i ./src/input.css -o ./dist/output.css --watch
```

### View the Website

Open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js http-server (install with: npm install -g http-server)
http-server

# Using VS Code Live Server extension
# Right-click on index.html and select "Open with Live Server"
```

## Performance Benefits

Using Tailwind CLI provides significant performance improvements:

- **File Size**: 1.7KB (minified) vs ~3.5MB (CDN)
- **Load Time**: ~10ms vs ~500ms+ on slow connections
- **Production Ready**: Only includes CSS classes actually used in your HTML
